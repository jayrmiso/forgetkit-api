# Decision Gate

Use this before architecture, dependency, compliance, security, privacy, licensing, auth, database, migration, folder-structure, API-contract, or product-direction decisions.

Required output:
- Decision needed
- Options
- Tradeoffs
- Recommended option
- Questions for the user

Do not implement until the user approves the direction.
